String and Number Interaction

If we add a tring to a number, the data type will be a string. If we try to subtract or perform invalid operations between non-numeric strings, it will give NaN, which stands for "Not a Number".

console.log("shahab" - "rasool");

If we have two true values, the result will be 2, because the default value of true is 1.

Expressions

An expression is a value that can be a number, string, boolean, null, undefined, object, array, or function. Expressions can be a single value or a combination of values. A combination of operands and operators is called an expression.

Operators

  • Assignment

    The equal sign (=) is called the assignment operator.

  • Arithmetic Operators

    These perform basic mathematical operations on variables or values, such as addition, subtraction, etc.

  • Modulus Operator

    The remainder operator (%) is called the modulus operator.

  • Comparison Operators

    Comparison operators are used to compare two values and return a Boolean result (true or false).

  • Strict Equality (===)

    It checks both the value and the type of the operands. If the types are different, it immediately returns false.

  • Ternary Operator

    A shorthand for an if-else statement, allowing you to perform simple conditional operations concisely.
